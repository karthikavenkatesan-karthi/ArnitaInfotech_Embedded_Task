# Air Quality Monitoring System using Arduino

## 📌 Project Description
This project monitors air quality using a gas sensor and displays pollution levels.

## 🛠 Components Used
- Arduino Uno
- Gas Sensor
- LCD Display

## ⚙️ Working
The gas sensor reads air quality values.
If the value exceeds a threshold, a warning message is displayed.
